### 迷你图片放大器

基于hammer.js的H5图片放大器，可放大缩小，可拖动。

使用方法：
```
  ImageViewer({
    url: 'https://......'
  })
```

| 参数       |  说明   |  备注   | 
| --------  | ----- |  -----   | 
| url     | 图片地址  |   必选   |
